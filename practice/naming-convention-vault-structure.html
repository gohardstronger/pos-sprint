01 Формула

{project} {type} описание – YYYY-MM-DD.md
{project} {type} описание – дата
Дата — в конце. Всегда. {project} {2026-03-04} тема — ломает сортировку.
EN DASH (–) — единственное тире. Mac: Option + -. Win: Alt+0150. Макс. 80 символов.

03 Проектные коды

2–8 символов. Уникальный. Стабильный.

{startup}       — основной бизнес
{product}       — конкретный продукт
{consulting}    — консалтинг
{channel}       — канал, блог
{course}        — образовательный проект
{self}          — личное развитие
Комбинации: {startup} {research} анализ рынка — проект + тип. Максимум 2 кода.

04 Структура Vault

Базовая

vault/
├── _inbox/          ← всё новое сюда
├── _templates/      ← шаблоны
├── projects/        ← по проектам
├── meetings/        ← транскрипты
├── rules/           ← правила
├── articles/        ← контент
└── resources/       ← справочные

Расширенная (200+ файлов)

vault/
├── _inbox/
├── _templates/
├── projects/
│   └── {startup}/
│       ├── artifacts/
│       ├── sources/
│       └── results/
├── meetings/
├── rules/
├── articles/
├── @people/         ← карточки (CRM)
├── daily/
└── archive/

6–10 папок на верхнем уровне. Flat > nested. Файл живёт по контексту, тип — в имени.

06 Теги

Дополняют, не дублируют имя файла. 3–5 на файл.

type/       ← тип (type/research)
project/    ← проект (project/startup)
topic/      ← тема (topic/automation)
status/     ← статус (status/active)
МеханизмДля чего
ПапкиФизическое расположение
ТегиНевидимые измерения, фильтрация
[[Ссылки]]Смысловые связи
Имя файлаКонтекст с первого взгляда

08 AI-агенты

AI-агент работает с vault как с кодовой базой. CLAUDE.md в корне vault — ваши правила для агента:

# Vault Rules

## Naming Convention
{project} {type} description – YYYY-MM-DD.md

## Active Projects
- {startup} — main business
- {consulting} — client projects

## File Types
- {research}, {guide}, {rule}, {plan}, {article}

## Folder Rules
- New files → _inbox/
- Never create new top-level folders
- Max 5 tags. YAML frontmatter always.

Работает с Claude Code, Codex (codex.md), Cursor (.cursorrules).
