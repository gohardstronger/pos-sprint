# POS Sprint

Коллекция правил, промптов и best practices для эффективной работы с AI-ассистентами (Claude Code, Cursor и др.).

## Что внутри

- **Правила оркестрации моделей** — как распределять задачи между Opus, Sonnet и Haiku для экономии токенов и повышения качества
- Таблица делегирования задач по моделям
- Паттерны автоэскалации при провале subagent'ов
- Правила сессий и антизависания

## Для кого

Для продвинутых пользователей Claude Code, которые хотят выжать максимум из мультимодельной оркестрации.

## Участники

Репозиторий открыт для участников [POS Sprint](https://github.com/ai-mindset-org/pos-sprint) — добавляйте свои правила и промпты через PR.
